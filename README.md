# api-demo-pratice
簡易的api範本練習

說明:
使用visual studio預設的範本
及內建的swagger介面，
所做的基本api串接練習，

分為兩個專案
◆ UsersListApi
◆ getUserFromApi

以及postman的json檔
◆ userApi.postman_collection

//////////////////////////////////////////////

● UsersListApi
用txt檔的讀寫功能模擬資料提取，
提供新增 編輯 刪除的web api功能，
建置後可用getUserFromApi進行模擬串接。

● getUserFromApi
用來串接UsersListApi專案
